Functional Programming with Groovy

03 August 2016 - Bill Fly

Abstract

At the June JavaMug meeting, Dave Thomas said “Object-Oriented Programming is Dead!” and Functional Programming is gaining popularity. So what is Functional Programming and how can we use it with Groovy. Groovy is not a pure functional language, we can use it in a functional style.
